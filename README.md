# Basic User Authentication

## What is this project?

- This is a mobile app that users can register accounts with email and password and login
- Their accounts are registered and recognized by Google Firebase Authentication
- Firebase will generate token for users to login



## How to Run this project?

- Download the project folder and run `npm install` to install all the packages
- Register your Firebase Authentication App, and change the token in the `auth.js` file in the `util` folder
- Open your mobile device simulators: Android Studio and Xcode both work
- Run `npm start` to start the `expo` with your simulator
